# ttpkgUnpacker

### 适用范围

可以解包抖音小程序的包和一些相同文件类型的包，包括包pkg文件（文件开头必须是TPKG）和ttpkg.js配置文件。

### 使用方法

1. 直接下载Releases中集成好的软件，可以直接输入需要解包的文件所在的**文件夹**的路径，就可以直接进行解包。

2. 将获得的包直接放入js文件夹中运行main就可以了。
